PeptideShaker

interpretation of proteomics identifications from multiple search engines

PeptideShaker is being developed by Marc Vaudel, Harald Barsnes and Lennart Martens.

PeptideShaker home page: http://compomics.github.io/projects/peptide-shaker.html.

Please cite PeptideShaker as: Vaudel et al.: Nature Biotechnol. 2015 Jan;33(1):22-24.

Supported Platforms

The tool has been developed on Windows XP and Windows 7. But should also work on older Windows versions and on other platforms, as long as Java 1.6 (or newer) is installed.
