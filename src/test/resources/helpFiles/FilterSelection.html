Filter Selection

PeptideShaker allows you to filter proteins, peptides and PSMs in two ways: (i) star the filtered values or (ii) hide the filtered values.

To add a filter simply locate the wanted filter level, i.e., protein, peptide or PSM, and click the Add button next to either the starred or hidden table option.

Edit existing filters by first selecting the filter and then click the corresponding Edit button.

When the desired filters have been set up and selected, click OK to apply the filters.
