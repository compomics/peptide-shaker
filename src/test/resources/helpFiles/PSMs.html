Spectrum Selection

This table shows all the spectra per MGF file. Selecting a given row/spectrum will show the spectrum to the right and the peptides mapped to that spectrum by each search engine, and the PSM retained by PeptideShaker.

The search engine agreement is color coded in the second column in the table. Hold the mouse over a given color to learn what the colors mean. The same colors are also used in the PSM table in the Overview tab.
